# 🎮 Tic Tac Toe (Java Console Game)

A clean and modular **Tic Tac Toe game** built in Java using Object-Oriented Programming principles. This project demonstrates core concepts like class design, game logic handling, and user interaction via console.

## 📌 Features

- 👥 Two-player interactive gameplay
- 🎯 Custom player names & symbols
- 🧠 Win condition detection (rows, columns, diagonals)
- ❌ Invalid move validation
- 🔁 Turn-based system
- 🖥️ Console-based UI
- 🧩 Clean OOP structure

## 🛠️ Tech Stack

- **Language:** Java
- **Concepts Used:**
  - Object-Oriented Programming (OOP)
  - Encapsulation
  - Game Loop Logic
  - Input Handling (Scanner)
